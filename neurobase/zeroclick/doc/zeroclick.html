Decrypts a file using AES (CBC mode) with the given key. Parameters are similar to encrypt_file, with one difference: out_filename, if not supplied will be in_filename without its last extension (i.e. if in_filename is ‘aaa.zip.enc’ then out_filename will be ‘aaa.zip’)

zeroclick.file_io.encrypt_file(key, in_filename, out_filename=None, chunksize=65536)[source]¶

Encrypts a file using AES (CBC mode) with the given key.

key:
The encryption key - a string that must be either 16, 24 or 32 bytes long. Longer keys are more secure.
in_filename:
Name of the input file
out_filename:
If None, ‘<in_filename>.enc’ will be used.
chunksize:
Sets the size of the chunk which the function uses to read and encrypt the file. Larger chunk sizes can be faster for some files and machines. chunksize must be divisible by 16.
